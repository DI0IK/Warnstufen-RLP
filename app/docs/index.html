API

URL: /api/v1/data

Erlaubte Optionen:

  • /api/v1/data
  • /api/v1/data?district=Gebiet
  • /api/v1/data?last=Tage
  • /api/v1/data?last=Tage&district=Gebiet

Iframe

URL: /iframe

Erlaubte Optionen:

  • /iframe?district=Gebiet
  • /iframe?district=Gebiet&Inzidenz=true

Beispiel:

<iframe src="/iframe?district=KS Landau i.d.Pf.&args=Inzidenz;Hospitalisierung;Intensivbetten;colors" frameborder="0"></iframe>
